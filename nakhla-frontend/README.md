# Nakhla Front-End Prototype

## Project Description
Nakhla is a React-based front-end prototype that helps users discover Saudi local brands through one centralized platform.

## Features
- Login and sign up
- Search and browse brands by category
- Brand profile page
- Add/remove favorites
- Submit ratings and reviews
- Brand owner dashboard
- Admin dashboard
- Manage verification requests and categories
